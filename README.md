# GrowthHub

GrowthHub is a personal finance application for tracking periods, category budgets, spendings, incomes, and dashboard insights.

## Current Stack
- Next.js + React + TypeScript
- Clerk authentication
- Supabase data layer

## Documentation
- App documentation: docs/APP_DOCUMENTATION.md
- Backend + DB build prompt: docs/BACKEND_BUILD_PROMPT.md

## Local Development

Run the frontend app:

```bash
pnpm dev
```

Open http://localhost:3000.
